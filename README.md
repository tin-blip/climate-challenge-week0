African Climate Analysis Project

 Project Overview
This project analyzes 10+ years of climate data (2015-2026) focusing on temperature and precipitation trends across five African nations: Ethiopia, Kenya, Nigeria, Sudan, and Tanzania.

 Tasks Completed
- Data Ingestion (Task 2): Successfully processed NASA POWER climate datasets.
- Data Cleaning: Transformed Julian dates into standard datetime objects and handled missing values.
- Visualization (Task 3): - Created daily temperature fluctuation plots.
    - Applied 30-day moving averages to identify long-term climate trends.
    - Built a 5-country comparison dashboard to analyze regional variances.
 
  Vulnerability Analysis Results
- Based on the integrated analysis of temperature extremes and precipitation totals:
    Highest Vulnerability: Sudan (High variability, lowest rainfall).
    Moderate Vulnerability: Nigeria and Kenya.
    Lowest Vulnerability (Most Resilient): Ethiopia (Consistently higher rainfall and stable temperatures).

 Insights
- Highest Temperatures: Sudan consistently recorded the highest temperature peaks.
- Stable Climates: Ethiopia showed the most temperate and stable climate profile.
- Rainfall: Identified clear correlations between seasonal humidity spikes and temperature drops.
